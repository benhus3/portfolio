Educational Sign Language Detection Game

A machine learning based, sign language detection game used to educate people on how to communicate using ASL.

Technologies used:

  • Keras CNN
  • Google Mediapipe
  • OpenCV
  • Matplotlib
  • Scikit-learn
  • Seaborn

I worked as part of a team containing three members, to create a model which could identify a number of different sign language gestures being made by the user and picked up by their webcam. This model was then used to create an educational game where the system would give the user a word to sign, and if the user correctly performed the symbol using their hands, they would gain a point.
